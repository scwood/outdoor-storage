Outdoor Storage inventory generator

Takes exports of the Unit and Customer tables and fills out the provided template file

Instructions:

  • In Access export the Unit table and the Customer table to seperate CSVs (make sure to include the headers).
  • Create a CSV template file where cells to be replaced have a space name in them (i.e. 01-00A01).
  • If a customer is renting that space it will be replaced in the template with the customer name. Empty spaces will be cleared.
